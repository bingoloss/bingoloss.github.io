常用用法说明

标题

以#符开头，共六级，一级就以一个#开头，二级就以两个#开头（写成##），其余依此类推。

标准 Markdown 的水平线

在标准 Markdown 中，一行由三个以上连续的减号组成的文本会被视为水平线。有些编辑器也支持星号符（*）和下划线字符（_）。
而在 LME 中还可以包含全角、半角空格。例如：

划分一行

Markdown加粗

以前后各两个 （星号）或各两个_（下划线符）包围的文本是加粗文本。
*加粗文本 或者 加粗文本

Markdown倾斜

以前后各一个*（星号）或各一个_（下划线符）包围的文本是倾斜文本。

Markdown块引用

块引用一般用于注释。
以>（大于号加空格）开头的行会被转译成html里的<blockquote<>/blockquote<，>符号后的空格可不要。
考虑到中文输入法频繁切换会很繁琐，所以用》或》　开头也是可以的。
其实用右单书名号开头（〉或〉　）也行。 空格也可以省略。

Markdown列表

列表有两种：无序列表、有序列表。

无序列表

以-、+、 这三个字符开头的行是无序列表项。这三个字符使用任意一个字符都可以，编译后的结果是一样的。
但是使用作为列表开头会与加粗、倾斜这两种效果冲突，所以尽可能不要用！！！如果实在必须在一行开头使用*来表示无序列表，要注意必须在后面跟一个空格（全、半角均可）；或者格式化一下。
注意：无序列表与加粗可能冲突！！！

有序列表

有序列表由数字开头，后跟句点（或．，也可以是、）。
注意避免时间等数字开头的句子可能会被误认为有序列表，此时： 1986.04，发生了一件事情… 这样来避免

Markdown链接

链接分几种：文字超链接、图像超链接、索引超链接。

2、以[= =]包围的文本会加上删除线。(有时候不行)

Markdown材料出处标记

如果一行文本以破折号开头，并紧跟一对书名号，则会被当作资料出处，所以在 CSS 中动了点手脚来支持它,在编译的 Html 文档中自动右对齐。
